# Place TTP team photos here when ready

Drop `.jpg` or `.png` files in this folder, then change the `src=` in `index.html` from:

```html
<img src="https://images.unsplash.com/photo-XXXX..." />
```

to:

```html
<img src="assets/img/your-photo.jpg" />
```

Recommended sizes:
- **Portraits** (the "Real humans" strip): 600×800px, JPEG, 80% quality
- **Service cards**: 900×500px (16:9), JPEG, 80% quality
